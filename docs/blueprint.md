# **App Name**: VibrantDev Portfolio

## Core Features:

- Dynamic Multi-Section Display: Showcase portfolio content across dedicated sections (Hero, Services, Portfolio, About, Contact, Footer) with smooth-scrolling navigation.
- Responsive & Adaptive Design: Ensure the entire website is fully responsive, mobile-first, and adapts seamlessly across all device screen sizes.
- Multilingual Content Switch: Allow users to toggle between English (US) and Portuguese (Brazil) content, powered by locale-specific JSON files.
- Interactive Portfolio Showcase: Present mock client projects in a dynamic grid with engaging hover effects and preview overlays.
- Direct Contact & Inquiry Form: Provide prominent access to contact methods (WhatsApp, Email, Instagram) and a simple, client-facing contact form for direct inquiries.
- Persistent Navigation & Action Button: Implement a sticky navigation bar that adapts visually on scroll and a floating Call-to-Action button for WhatsApp.

## Style Guidelines:

- Primary color (accent): A vibrant, techy purple (#6F23F6) to represent creativity and professionalism in a dark environment.
- Background color: A deep, subtly purple-tinted charcoal (#28222d) provides a modern dark theme foundation.
- Accent color: A bright, electric blue (#78ADFF) to provide visual contrast, highlight interactive elements, and enhance the futuristic feel.
- Headline font: 'Space Grotesk' (sans-serif) for its modern, slightly futuristic, and tech-inspired aesthetic.
- Body font: 'Inter' (sans-serif) for clean readability and a contemporary, objective look that pairs well with 'Space Grotesk'.
- Utilize sleek, modern line-art icons, potentially with subtle neon glow effects, especially in the Services section, aligning with the futuristic dark theme.
- Sections are cleanly separated with ample whitespace, using glassmorphism effects for cards and containers, and subtle soft shadows to create depth in a minimal dark interface.
- Implement smooth scroll-triggered animations (e.g., fade-ins, slides, subtle parallax) for section reveals and content loading, combined with interactive hover effects on cards and portfolio items.
